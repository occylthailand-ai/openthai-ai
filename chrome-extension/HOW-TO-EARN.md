# วิธีรับเงินจริง — ไม่มีโกหก

## 3 ขั้นตอน คลิกเดียวชีวิตเปลี่ยน

### ขั้นที่ 1: ติดตั้งส่วนขยาย
1. เปิด Chrome → พิมพ์ `chrome://extensions`
2. เปิด "Developer mode" (มุมขวาบน)
3. คลิก "Load unpacked" → เลือกโฟลเดอร์ `chrome-extension`

### ขั้นที่ 2: ลงทะเบียน Affiliate (ฟรี ทำครั้งเดียว)

| Platform | ลิงก์สมัคร | รายได้ |
|----------|-----------|-------|
| **Shopee** | affiliate.shopee.co.th | 2–10% / ยอดขาย |
| **Lazada** | affiliate.lazada.co.th | 2–8% / ยอดขาย |
| **AliExpress** | portals.aliexpress.com/affportals | 3–9% / ยอดขาย |

หลังสมัคร → เอา Code มาใส่ใน "ตั้งค่า" ของส่วนขยาย

### ขั้นที่ 3: ใช้งานทุกวัน

```
1. เปิดหน้าสินค้าบน Shopee / Lazada
2. คลิกไอคอน Openthai.ai ใน Chrome
3. กด "สร้างคอนเทนต์ + Affiliate Link"
4. คัดลอกไปโพสต์ TikTok / Facebook / LINE
5. รอคนคลิกลิงก์และสั่งซื้อ → เงินเข้า
```

## ความจริงที่ต้องรู้

- คอมมิชชั่นเข้าบัญชีตาม **เงื่อนไขของ Shopee/Lazada** (ไม่ใช่ระบบเรา)
- ต้องมี Follower หรือกลุ่มเป้าหมายจริง — ไม่มีใครคลิก = ไม่มีรายได้
- AI สร้างคอนเทนต์ให้ **แต่คุณต้องโพสต์เอง**
- รายได้ขึ้นกับสินค้า, ราคา, และความสม่ำเสมอในการโพสต์
