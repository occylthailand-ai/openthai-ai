# OpenThai AI — OTOP Partner Outreach Templates
# เทมเพลตติดต่อพาร์ทเนอร์ OTOP 10 รายแรก

ใช้หลัก Authentic Communication: เรียกชื่อ, แสดงว่าศึกษามาก่อน, ให้ก่อนขอ, หา Common Ground

---

## วิธีใช้เทมเพลตนี้

1. **ค้นหาก่อนติดต่อ** — รู้จักสินค้าเขาก่อน (Facebook Page, เว็บ OTOP พัฒนาชุมชน)
2. **เติม [ชื่อ] และ [สินค้า]** — ห้ามส่ง template เดิมทุกคน
3. **ส่งทีละคน** — รอ reply ก่อนส่งรายต่อไป
4. **Follow-up 1 ครั้ง** หลัง 3 วันถ้าไม่มี response

## Template A — LINE / Facebook Message (สั้น กระชับ เหมาะมือถือ)

> ใช้กับ: ผู้ผลิต OTOP ที่เจอผ่าน Facebook หรือ OTOP Online

```
สวัสดีครับ คุณ[ชื่อ] 🙏

ผมชื่อ [ชื่อคุณ] จาก OpenThai AI ครับ

เห็น[สินค้า]ของคุณแล้วประทับใจมากครับ โดยเฉพาะ[จุดเด่นที่เห็น เช่น: กรรมวิธีการทอมือ / สูตรโบราณ / วัตถุดิบท้องถิ่น]

เราทำเครื่องมือ AI ช่วยสร้างคอนเทนต์ TikTok ให้ผู้ผลิต OTOP ครับ — พิมพ์ชื่อสินค้า ได้สคริปต์+แคปชั่น+แฮชแท็กทันที

อยากให้คุณ[ชื่อ]ลองใช้ฟรีก่อนเลยครับ ไม่มีเงื่อนไข
→ [ลิงก์ generator.html]

ถ้าสะดวก รบกวนแจ้งด้วยนะครับว่าตอนนี้ขายช่องทางไหนเป็นหลัก จะได้แนะนำตรงๆ ครับ 🙏
```

## Template B — Email ทางการ (สำหรับ OTOP ที่มีอีเมล หรือ ผ่านองค์กร)

> ใช้กับ: SME / OTOP ที่มีเว็บไซต์หรือติดต่อผ่านช่องทางธุรกิจ

**หัวข้อ:** `[ชื่อสินค้า] × OpenThai AI — ทดลองใช้ฟรี 30 วัน ไม่มีเงื่อนไข`

```
เรียน คุณ[ชื่อ] ครับ/ค่ะ

ผม[ชื่อคุณ] จาก OpenThai AI ครับ — เครื่องมือ AI สร้างคอนเทนต์ TikTok สำหรับสินค้าไทย

ผมศึกษา[สินค้า]ของคุณมาก่อนติดต่อครับ และพบว่า[จุดที่น่าสนใจ เช่น: 
"สินค้ามีเรื่องราวที่แข็งแกร่งมากเรื่องภูมิปัญญาชุมชน / 
กระบวนการผลิตที่น่าทึ่งซึ่งคนทั่วไปไม่เคยเห็น /
วัตถุดิบที่หายากและมีเอกลักษณ์เฉพาะพื้นที่"]

สิ่งเหล่านี้คือ "Gold Content" บน TikTok ครับ — แต่ผู้ผลิตส่วนใหญ่ยังไม่รู้จะเล่าเรื่องยังไง

─────────────────────────────
OpenThai AI ช่วยได้:
✅ พิมพ์ชื่อสินค้า → ได้สคริปต์ 30 วินาทีทันที
✅ AI เลือก Hook ที่เหมาะกับสินค้าของคุณ
✅ ได้แคปชั่น + 10 แฮชแท็กพร้อมใช้
✅ AI อธิบายว่าทำไม content นี้ถึงน่าจะ viral
─────────────────────────────

ขอเสนอ: ทดลองใช้ฟรี 30 วัน ไม่ต้องใส่บัตรเครดิต
→ [ลิงก์ generator.html]

หรือถ้าต้องการ Demo สด ผมยินดี Video Call 15 นาทีครับ
เพียงตอบ email นี้หรือ LINE: [LINE ID ของคุณ]

ขอบคุณที่ทุ่มเทสร้างสินค้าไทยดีๆ ครับ 🙏

[ชื่อคุณ]
OpenThai AI
openthai-ai.com
```

## Template C — ติดตาม Follow-up (3 วันหลังส่ง Template A หรือ B)

```
สวัสดีครับ คุณ[ชื่อ] 🙏

ผม[ชื่อคุณ] จาก OpenThai AI ส่ง message มาเมื่อ 3 วันก่อนครับ

เข้าใจว่าคงยุ่งมากครับ แค่อยากให้รู้ว่า offer ทดลองใช้ฟรียังเปิดอยู่นะครับ

ระหว่างนี้ผมลองสร้าง content ตัวอย่างสำหรับ[สินค้า]ของคุณมาให้เลยครับ:

---
🎬 [Hook ตัวอย่างที่สร้างด้วย OpenThai AI]
"[เขียน Hook จริงๆ ที่เกี่ยวกับสินค้าเขา เช่น: รู้ไหมว่าผ้าไหมมัดหมี่ผืนนึง ใช้เวลาทอนานกว่า 3 อาทิตย์?]"
---

ถ้าสนใจอยากเห็นสคริปต์เต็มๆ แจ้งได้เลยครับ ทำให้ฟรีครับ 🙏
```

## หลักการ Authentic Communication (สรุปสั้น)

| ❌ อย่าทำ | ✅ ให้ทำ |
|---|---|
| Copy-paste template เดิมทุกคน | เรียกชื่อ + พูดถึงสินค้าเขาจริงๆ |
| ขายทันทีตั้งแต่ message แรก | ให้ก่อน (content ตัวอย่าง, ทดลองฟรี) |
| พูดถึงตัวเองและ feature มาก | ถามถึงปัญหาและความต้องการเขา |
| ส่ง mass message ทีเดียว 100 คน | ส่งทีละ 5-10 คน ดูแลอย่างละเอียด |
| หายไปหลัง reply ครั้งแรก | Follow-up สม่ำเสมอ ปรากฏตัวยามจำเป็น |
