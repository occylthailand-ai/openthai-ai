## เวอร์ชันยาว (สำหรับครีเอเตอร์/นักรีวิวที่รู้จัก)

หากคุณมีช่องทางโซเชียล (TikTok / IG / YouTube / Facebook) และสนใจสร้างรายได้
จากการแนะนำสินค้าไทย OpenThaiAi เปิดโปรแกรม Affiliate ที่มี 3 ระดับ:
- Starter — 10% เริ่มต้นได้ทันทีไม่มีขั้นต่ำ
- Pro — 20% เมื่อยอดขายสะสมถึง 50,000 บาท
- Elite — 30% เมื่อยอดขายสะสมถึง 200,000 บาท

มี Dashboard ติดตามยอดขายแบบ real-time ลิงก์ referral ส่วนตัว และระบบจ่ายเงิน
อัตโนมัติ สมัครได้ที่: https://www.openthai-ai.com/portals/affiliate

## หมายเหตุการใช้งาน
เช่นเดียวกับไฟล์อื่นในโฟลเดอร์นี้ — ส่งถึงคนที่รู้จักจริงหรือโพสต์ในพื้นที่
สาธารณะที่เปิดให้ประชาสัมพันธ์ ไม่ scrape โปรไฟล์/เบอร์ติดต่อจากที่ใดมาโดยพลการ

## หมายเหตุเรื่อง "สินค้า" (บัญญัติข้อ 4)
ระบบนี้ไม่มีฟอร์มสมัคร "สินค้า" แยกต่างหาก — สินค้าเข้าสู่แพลตฟอร์มผ่านการ
สมัครของผู้ผลิต (`/portals/producer` → `backend/inventory.js`) ผู้ผลิตแต่ละราย
เพิ่มสินค้าของตัวเองหลังได้รับอนุมัติ ดังนั้นการ "เพิ่มจำนวนสินค้า" ในทางปฏิบัติ
คือผลลัพธ์ของการเพิ่มจำนวนผู้ผลิตจริง (ดู producer-recruitment-post.md) ไม่ใช่
กระบวนการแยกที่ต้องสร้างเพิ่ม
