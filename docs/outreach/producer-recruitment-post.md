## หมายเหตุการใช้งาน

- ส่งเฉพาะกับคนที่คุณติดต่อได้จริงหรือกลุ่มสาธารณะที่เปิดให้โพสต์ประชาสัมพันธ์
  (ไม่ใช่การส่งหาเบอร์/อีเมลที่เก็บมาโดยไม่ได้รับความยินยอม)
- ทุกคนที่สมัครผ่านฟอร์มนี้จะเก็บข้อมูลจริงใน `portal_leads` table
  (ตรวจสอบได้ที่ Admin Panel แท็บ "📨 เชิญผู้ผลิต")
- เนื้อหาเดียวกันนี้ปรับใช้กับ `/portals/consumer` และ `/portals/middleman`
  ได้ในลักษณะเดียวกัน โดยเปลี่ยนสิทธิประโยชน์ให้ตรงกับฟอร์มนั้นๆ
