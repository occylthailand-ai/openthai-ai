# AI Copywriting Templates — OTOP (สำหรับทีม Affiliate)

Template ปิดการขายพร้อมใช้ทันที ไม่ต้องรอ AI generate — ครอบคลุมข้อความสั้น (โพสต์/แชท/คอมเมนต์)
และสคริปต์ไลฟ์แบบยาว แยกตามหมวดสินค้า

รวมเข้ากับ `docs/outreach/affiliate-sales-scripts.md` (เวอร์ชันสั้น hook+CTA เดียว) —
ไฟล์นี้คือเวอร์ชันขยาย มี AIDA framework + Live Selling Script แบบเต็ม

---

## 1. คำแนะนำการใช้งาน

1. **ทุก placeholder ใน `[ ]` ต้องแทนที่ด้วยข้อมูลสินค้าจริงก่อนใช้งาน** ห้ามส่ง/พูด
   ทั้งที่ยังมี `[ ]` ค้างอยู่
2. **ห้ามใส่ตัวเลข ยอดขาย จำนวนรีวิว หรือคำรับรองที่ไม่มีที่มาจริง** ถ้าไม่มีข้อมูลจริง
   ให้ตัดประโยคนั้นออก ไม่ใช่ใส่ตัวเลขสมมติ — กฎเดียวกับทุกไฟล์ใน `docs/outreach/`
3. Template เหล่านี้เป็น**จุดเริ่มต้น** ไม่ใช่คำตอบสุดท้าย — ปรับสำนวนให้เข้ากับสไตล์ผู้ขาย/
   แพลตฟอร์มที่ใช้จริง (ข้อความแชทกับสคริปต์ไลฟ์ใช้โทนต่างกันได้)
4. หากต้องการเวอร์ชันที่ AI เขียนให้เฉพาะสินค้าจริง (ใส่ชื่อ/ราคา/จุดเด่นอัตโนมัติ) ระบบมี
   2 สกิลที่ทำงานจริงอยู่แล้ว ไม่ต้องสร้างใหม่:
   - **S18 Sales Conversion Engine** — `POST /api/skills/promo-engine` (ข้อความขาย/โพสต์)
   - **S25 Live Selling Script** — `POST /api/skills/live-script` (สคริปต์ไลฟ์ TikTok/FB/Shopee Live)

## 2. โครงสร้าง AIDA (ใช้เป็นแกนของทุก Template)

| ขั้นตอน | หน้าที่ | ตัวอย่างสั้น |
|---|---|---|
| **A** — Attention (ดึงความสนใจ) | บรรทัดแรกต้องหยุดคนเลื่อนฟีด | คำถาม/สถิติ/ปัญหาที่คนเจอบ่อย |
| **I** — Interest (สร้างความสนใจ) | อธิบายสินค้า/จุดเด่นที่ต่างจากเจ้าอื่น | วัตถุดิบ/กรรมวิธี/ที่มา |
| **D** — Desire (กระตุ้นความอยาก) | ทำให้รู้สึกว่า "ต้องมี" ไม่ใช่แค่ "น่าสนใจ" | แก้ปัญหาอะไรให้ลูกค้า / ใช้แล้วชีวิตดีขึ้นยังไง |
| **A** — Action (ปิดการขาย) | บอกให้ทำอะไรต่อ ชัดเจน มี urgency | ลิงก์ + เหตุผลให้ตัดสินใจตอนนี้ |

ข้อมูลจริงจากระบบ (`S9 Learning Layer` — วิเคราะห์จาก feedback จริงในแพลตฟอร์ม)
ยืนยันว่าเนื้อหาที่มี **hook (Attention) + urgency (Action)** ชัดเจน ทำคะแนนได้สูงสุด —
ทุก template ด้านล่างจึงเน้น 2 จุดนี้เป็นพิเศษ

## 3. Template แยกตามหมวดหมู่

### 🍽️ อาหารและเครื่องดื่ม (Food & Beverage)

**AIDA:**
- **A:** ของกินที่ [จุดขาย เช่น อร่อยจนขายหมดไว/หากินยาก] แบบนี้ยังไม่เคยลองเหรอคะ?
- **I:** ทำจาก [วัตถุดิบหลัก] ด้วยกรรมวิธี [เช่น อบ/หมัก/สกัดเย็น] ไม่ใส่ [สิ่งที่หลีกเลี่ยง]
- **D:** เหมาะกับ [โอกาส/กลุ่มเป้าหมาย เช่น ของฝาก/มื้อว่างเพื่อสุขภาพ] เก็บได้ [ระยะเวลา] พกพาสะดวก
- **A:** วันนี้ [โปรโมชั่นถ้ามีจริง] 👉 [ลิงก์ affiliate] จำนวนจำกัดต่อรอบผลิต

**Live Selling Script (ยาว):**
> "สวัสดีค่ะทุกคน วันนี้เอา [ชื่อสินค้า] มาแนะนำนะคะ อันนี้คือ [คำอธิบายสั้นว่าคืออะไร]
> ทำจาก [วัตถุดิบหลัก] แบบ [กรรมวิธี] เลยค่ะ ไม่ใส่ [สิ่งที่หลีกเลี่ยง] เลย
>
> จุดเด่นของตัวนี้คือ [จุดขายหลัก 1-2 ข้อ] ใครที่กำลังมองหา [โอกาสใช้งาน] อยู่
> อันนี้ตอบโจทย์มากค่ะ เก็บได้ [ระยะเวลา] พกไปไหนก็ได้
>
> ผลิตโดย [ชื่อกลุ่มผู้ผลิต/แหล่งที่มา ถ้ามีข้อมูลจริง] [รายละเอียดมาตรฐาน ถ้ามี เช่น อย./OTOP]
>
> ตอนนี้กดลิงก์ในคอมเมนต์/ไบโอได้เลยนะคะ [รายละเอียดโปรโมชั่นถ้ามีจริง]
> ล็อตนี้มีจำนวนจำกัด ใครสนใจรีบตัดสินใจเลยค่ะ"

### 🌿 สมุนไพรและผลิตภัณฑ์เพื่อสุขภาพ (Herbal & Health)

**AIDA:**
- **A:** ผิว/สุขภาพ [ปัญหาที่พบบ่อย] ยังหาตัวช่วยไม่ได้อยู่เหรอคะ?
- **I:** สกัดจาก [วัตถุดิบหลัก] แบบ [กรรมวิธี] ไม่ใส่ [สารที่หลีกเลี่ยง]
- **D:** ใช้แล้วช่วย [ประโยชน์ที่แก้ปัญหาตรงจุด] เหมาะกับคนที่ [กลุ่มเป้าหมาย]
- **A:** สั่งวันนี้ [โปรโมชั่นถ้ามีจริง] 👉 [ลิงก์ affiliate]

**Live Selling Script (ยาว):**
> "ใครมีปัญหาเรื่อง [ปัญหาที่สินค้าตอบโจทย์] บ้างคะ ยกมือในคอมเมนต์ได้เลย
> วันนี้แอดมินเอา [ชื่อสินค้า] มาให้ดูกันค่ะ เป็นสมุนไพรไทยแท้ สกัดจาก [วัตถุดิบหลัก]
> ด้วยกรรมวิธี [เช่น สกัดเย็น/หมักธรรมชาติ] ไม่ใส่ [สารกันเสีย/สารเคมีที่หลีกเลี่ยง] เลยนะคะ
>
> ตัวนี้ผลิตโดย [กลุ่มวิสาหกิจชุมชน/แหล่งที่มาจริง] [มาตรฐานที่มีจริง เช่น อย./GMP]
> ใช้ต่อเนื่องแล้วจะช่วย [ประโยชน์ที่แท้จริงตามฉลาก ห้ามอวดอ้างเกินจริง]
>
> ราคาวันนี้ [ราคา/โปรถ้ามี] กดลิงก์ในคอมเมนต์ได้เลยค่ะ ของมีจำนวนจำกัดในแต่ละรอบผลิตนะคะ"

### 💇 ความงามและการดูแลร่างกาย (Beauty & Personal Care)

**AIDA:**
- **A:** อยาก [ผลลัพธ์ที่ลูกค้าต้องการ] แบบไม่ทำร้ายผิว/ผมเหรอคะ?
- **I:** ส่วนผสมหลักคือ [ระบุ] เหมาะกับ [สภาพผิว/ผมเป้าหมาย]
- **D:** ใช้เป็นประจำแล้วช่วย [ผลลัพธ์ตามจริงที่ระบุบนฉลาก]
- **A:** ลิงก์สั่งซื้อ 👉 [ลิงก์ affiliate] [โปรโมชั่นถ้ามีจริง]

**Live Selling Script (ยาว):**
> "ใครกำลังหาตัวช่วยเรื่อง [ปัญหาผิว/ผม] อยู่บ้างคะ วันนี้มี [ชื่อสินค้า] มาแนะนำ
> ส่วนผสมหลักคือ [ระบุ] เหมาะกับคนที่มีปัญหา [กลุ่มเป้าหมาย] โดยเฉพาะเลยค่ะ
>
> [อธิบายวิธีใช้สั้นๆ ถ้าจำเป็น] ใช้ต่อเนื่อง [ระยะเวลาที่แนะนำ] จะเห็นผลเรื่อง [ผลลัพธ์ตามฉลากจริง]
>
> เป็นสินค้า OTOP จาก [แหล่งผลิตจริง] [ถ้ามีการรับรองจริงใส่ตรงนี้]
>
> วันนี้กดลิงก์สั่งได้เลยค่ะ [โปรโมชั่นถ้ามีจริง] ส่วนลดพิเศษเฉพาะผ่านลิงก์นี้เท่านั้นนะคะ"

### 🧵 ผ้าและเครื่องแต่งกาย (Textile & Apparel)

**AIDA:**
- **A:** งานฝีมือแท้ๆ แบบนี้หายากขึ้นทุกวันนะคะ ใครชอบของไม่ซ้ำใครบ้าง?
- **I:** [เทคนิคการทอ/ตัดเย็บ/ลวดลาย] จากช่างฝีมือ [แหล่งที่มา]
- **D:** ใส่แล้วมีชิ้นเดียวในโลก ไม่ซ้ำแบบ เหมาะกับ [โอกาสใช้งาน]
- **A:** มีจำนวนจำกัดต่อรอบผลิต 👉 [ลิงก์ affiliate]

**Live Selling Script (ยาว):**
> "ดูผ้าผืนนี้ก่อนนะคะ [อธิบายลวดลาย/สีสัน] ทอด้วยมือทั้งผืนเลยค่ะ
> เป็นงานฝีมือจาก [ชื่อกลุ่มช่างทอ/ชุมชน] สืบทอดกันมา [ถ้ามีประวัติจริงใส่ตรงนี้]
>
> เทคนิคที่ใช้คือ [เทคนิคการทอ] ทำให้แต่ละผืนไม่เหมือนกันเป๊ะๆ เลยนะคะ
> ใส่ไปงาน [โอกาสใช้งาน] ได้เลย เท่มาก ไม่ซ้ำใครแน่นอน
>
> ล็อตนี้ผลิตได้จำนวนจำกัดค่ะ เพราะทอมือทุกผืน ใครสนใจกดลิงก์จองไว้ก่อนได้เลยนะคะ"

### 🏺 หัตถกรรมและของใช้ในบ้าน (Handicraft & Home)

**AIDA:**
- **A:** อยากได้ของแต่งบ้านที่มีเรื่องราว ไม่ใช่ของโรงงานเหรอคะ?
- **I:** ทำจาก [วัสดุ] โดยช่างฝีมือ [แหล่งที่มา] ด้วยเทคนิค [ระบุ]
- **D:** ใช้ได้จริง [ประโยชน์การใช้งาน] แถมมีเรื่องราวความเป็นมาที่ของโรงงานให้ไม่ได้
- **A:** จองก่อนได้คิวผลิตก่อน 👉 [ลิงก์ affiliate]

**Live Selling Script (ยาว):**
> "ชิ้นนี้คือ [ชื่อสินค้า] ทำจาก [วัสดุ] โดยช่างฝีมือจาก [ชุมชน/แหล่งที่มา] นะคะ
> ใช้เทคนิค [เทคนิคการผลิต] ทุกชิ้นทำมือ ไม่ใช่งานโรงงาน
>
> เอาไปใช้ได้จริงในบ้านนะคะ [การใช้งานจริง] หรือจะซื้อเป็นของฝาก/ของตกแต่งก็ได้
>
> เพราะทำมือทุกชิ้น กำลังการผลิตต่อรอบมีจำกัดค่ะ ใครสนใจกดลิงก์จองคิวไว้เลยนะคะ"

### 🐾 อาหารสัตว์เลี้ยง (Pet Food) — หมวดใหม่

> **หมายเหตุ:** หมวดนี้เพิ่งเพิ่มเข้า dropdown จริงของ `/portals/consumer`
> (`CATEGORIES` ใน `ConsumerPortalPage.jsx`) พร้อมกับไฟล์นี้ — ก่อนหน้านี้ระบบยังไม่มี
> หมวดสัตว์เลี้ยงให้เลือกเลย

**AIDA:**
- **A:** น้องหมาน้องแมวที่บ้านกินอาหารเดิมๆ จนเบื่อหรือยังคะ?
- **I:** ทำจาก [วัตถุดิบหลัก] [กรรมวิธี] ไม่ใส่ [สารกันเสีย/สารเติมแต่งที่หลีกเลี่ยง]
- **D:** ช่วยเรื่อง [ประโยชน์ต่อสุขภาพสัตว์เลี้ยงตามจริง เช่น ขนเงางาม/ย่อยง่าย] เหมาะกับสัตว์เลี้ยงวัย [ระบุ]
- **A:** สั่งวันนี้ 👉 [ลิงก์ affiliate] [โปรโมชั่นถ้ามีจริง]

**Live Selling Script (ยาว):**
> "ทาสหมาทาสแมวคนไหนกำลังหาอาหาร/ขนมให้น้องบ้างคะ วันนี้เอา [ชื่อสินค้า] มาแนะนำ
> ทำจาก [วัตถุดิบหลัก] [กรรมวิธี] ไม่ใส่ [สารที่หลีกเลี่ยง] เลยค่ะ ปลอดภัยกับน้องแน่นอน
>
> เหมาะกับน้องวัย [ช่วงวัย] [ถ้ามีข้อมูลจริงเรื่องสายพันธุ์/ขนาดที่เหมาะใส่ตรงนี้]
> ช่วยเรื่อง [ประโยชน์ตามฉลากจริง เท่านั้น ห้ามอวดอ้างสรรพคุณทางการแพทย์]
>
> ราคาวันนี้ [ราคา/โปรถ้ามี] กดลิงก์สั่งให้น้องได้เลยค่ะ"

>
> [อธิบายวิธีเริ่มใช้งานสั้นๆ] เหมาะกับ [กลุ่มเป้าหมาย เช่น ผู้ประกอบการรายย่อย/ฟรีแลนซ์]

> วันนี้ [เงื่อนไข/โปรโมชั่นถ้ามีจริง] กดลิงก์เริ่มใช้งานได้เลยค่ะ"

## 4. Template ทั่วไป (ใช้ได้ทุกหมวด)

**AIDA:**
- **A:** [คำถาม/ปัญหาที่กลุ่มเป้าหมายเจอบ่อย]?
- **I:** [ชื่อสินค้า] คือ [อธิบายสั้นว่าคืออะไร] จุดเด่นคือ [1-2 ข้อ]
- **D:** เหมาะกับ [กลุ่มเป้าหมาย] เพราะช่วย [ประโยชน์หลัก]
- **A:** 👉 [ลิงก์ affiliate] [เหตุผลให้ตัดสินใจตอนนี้]

**Live Selling Script (ยาว):**
> "สวัสดีค่ะทุกคน วันนี้เอา [ชื่อสินค้า] มาแนะนำนะคะ
>
> [อธิบายว่าสินค้า/บริการนี้คืออะไร แก้ปัญหาอะไรให้ลูกค้า]
>

## หมายเหตุปิดท้าย

ไฟล์นี้เป็น static template ใช้ได้ทันทีไม่ต้องเรียก API — สำหรับสินค้าที่ต้องการความ
เจาะจงมากกว่านี้ (เช่น มีจุดขายเฉพาะที่ template ไม่ครอบคลุม หรือขี้เกียจแก้ placeholder เอง)
ใช้ **S18** (ข้อความ/โพสต์) หรือ **S25** (สคริปต์ไลฟ์) ซึ่ง generate ให้ตรงกับสินค้านั้นจริง
