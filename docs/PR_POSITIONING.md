## 3) Message Pillars (3 เสาหลัก — ทุกคอนเทนต์ควรอิงอย่างน้อย 1 เสา)

### เสา A — 🇹🇭 "Thai-first จริง ไม่ใช่แค่แปล"
เข้าใจตลาดนัด, ไลฟ์ขายของ, สินค้าชุมชน · สคริปต์ไทยพร้อมขายทันที
> proof: 5 รูปแบบ Hook ภาษาไทย, 300+ สินค้า OTOP/ไทย/โลก

### เสา B — 🧠 "สอนด้วย ไม่ใช่แค่สร้าง" *(ตัวชูโรง — ไม่มีคู่แข่งไหนมี)*
AI Critic ให้คะแนน ≥7/10 + Learning Layer อธิบายว่าทำไม Hook เวิร์ก
> proof: ผู้ใช้เก่งขึ้นเอง ไม่ผูกติดเครื่องมือ

### เสา C — 💸 "ครบวงจร + เข้าถึงได้"
สร้าง → ขาย → เก็บเงิน ในที่เดียว ราคาถูกกว่าฝรั่ง 5–10 เท่า
> proof: PromptPay/บัตร, สต๊อก, affiliate, เครดิต · ฿0/149/299
