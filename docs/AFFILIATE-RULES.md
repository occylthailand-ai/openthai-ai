## Affiliate Onboarding Flow
1. สมัคร → KYC (ชื่อ + บัตรประชาชน + บัญชีธนาคาร)
2. ยืนยัน Non-MLM Agreement (sign digital)
3. ได้รับ Referral Link + Dashboard
4. ทำ Sale ครั้งแรก → Activate Commission
5. จ่าย Commission ทุกวันที่ 1 และ 15 ของเดือน
