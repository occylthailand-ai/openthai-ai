Generate Thai social media content for a product using the openthai-ai MCP tool.

Use the `generate_content` tool (from the openthai-ai MCP server) with this input:
- product: $ARGUMENTS
- platform: TikTok
- style: sales
- lang: ภาษาไทย

Then present the result clearly with:
1. **Hook** (ประโยคเปิด)
2. **Script** (สคริปต์)
3. **Caption** (แคปชั่น)
4. **Hashtags** (แฮชแท็ก)

If the backend is offline, generate the content yourself following Thai TikTok best practices: short punchy hook (≤7 words), 3-step script (problem → solution → CTA), 5–8 relevant Thai hashtags.
